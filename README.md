# Arnau Vila Custom Search Engine

About
Built a search engine with a custom data structure that stores the word, word occurrences, and location of the word. The search engine is a full-stack single-thread or multi-threaded in memory web crawler that uses a work queue to build an index from a seed URL and a search engine web interface that uses embedded Jetty and servlets to search the index. A custom user interface was created by me in html and CSS.
